About Dusknoir

I play a Shadow priest and rogue. I've been playing since 2007, and since then I've went by both Dusknoir and Burial.

I love programming in Ruby and Lua and all of my projects are released into the public domain. You can find a list of my projects (some of which are Warcraft addons) on my GitHub page. If you need to get in touch, you can email me directly.

About This Site

This website is built with Middleman and hosted on GitHub Pages. Backgrounds are from Subtle Patterns (inverted Light Grunge and darkened Black Mamba — which looks best in Firefox, for some reason) and the fonts are Broken 15 and Droid Sans. These works are not my property.

Statistics are tracked by Google and comments are served by Disqus. Feel free to disable JavaScript if you're concerned with privacy; the only loss of functionality will be comments.
